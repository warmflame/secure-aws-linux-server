# Secure AWS Linux Server Deployment

## Project Overview

This project focuses on deploying and securing a cloud hosted Ubuntu server in AWS. The goal is to gain hands on experience with AWS infrastructure, Linux administration, Docker deployment, and basic cloud security practices.

## Objectives

- Deploy an Ubuntu EC2 instance in AWS
- Configure security group rules for controlled access
- Connect to the server using SSH
- Update and harden the Linux system
- Configure the UFW firewall
- Deploy a containerized web server using Docker
- Document security decisions, screenshots, and lessons learned

## Tools and Technologies

- AWS EC2
- Ubuntu Linux
- SSH
- UFW Firewall
- Docker
- Nginx
- GitHub

## Basic Architecture

User → AWS Security Group → EC2 Ubuntu Server → Docker Container → Nginx Web Page
